## 주제 선정 이유

- React, Redux 에 관해 학습한 내용을 바탕으로 반응형 뷰를 구성하는 경험하고자 함.
- 프로젝트매니징에 직/간접적으로 참여하면서, 서버/데이터베이스 구축, 배포까지의 경험하면서 성장하고자 함.
